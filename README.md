# TODO planner
> A planning tool that's supposed to keep me sane throughout my PhD.

## Setup
```
conda create -n todo_planner
conda activate todo_planner
conda install -c conda-forge fastapi uvicorn
```

## Usage
```
uvicorn main:app --reload
```
